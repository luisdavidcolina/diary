# diary

Esta es una aplicación web diseñada para la organización personal, profesional y educativa. 

## Objetivo
El propósito principal de esta aplicación es gestionar pendientes (tareas) y estructurar objetivos en distintas áreas de la vida del usuario (laboral, personal y educativa), como por ejemplo el seguimiento de temas de aprendizaje para materias específicas. 

La estructura y organización de los datos se irá refinando de forma progresiva a medida que se integren los datos y requisitos reales.

## Prioridades Actuales

> **⏳ Plazo Máximo:** Todo el plan de estudio y dominio de estos temas debe cumplirse **antes de septiembre** (un lapso aproximado de 2 meses).

### 1. Preparación para Examen de Suficiencia de Inglés
**Perfil del estudiante:** Principiante.
**Estrategia de estudio:**
- **Fase 1 (Enfoque en el Modelo):** Estudiar y practicar basándose directamente en el modelo del examen (ya que suelen repetir la estructura o contenido).
- **Fase 2 (Dominio del Tema):** Profundizar en todo el vocabulario, gramática y temas relacionados para alcanzar un dominio real del idioma desde el nivel principiante.

### 2. Cálculo Científico
- **Estado actual:** El primer parcial ya fue presentado.
- **Objetivo:** Preparación intensiva para los dos parciales restantes.

### 3. Seguridad en Redes
- **Objetivo:** Estudiar y dominar los conceptos fundamentales de la materia de Seguridad en Redes.

### 4. Computación en la Nube
- **Objetivo:** Estudiar y dominar los conceptos teóricos y prácticos de Computación en la Nube.

## Arquitectura de la Aplicación (App Web)
- **Stack Tecnológico:** React (mediante Vite) para un desarrollo rápido y estructurado basado en componentes.
- **Estilos y Diseño:** CSS puro (Vanilla CSS) enfocado en un diseño moderno, premium y oscuro (Dark Mode + Glassmorphism).
- **Base de Datos (BaaS):** Firebase Cloud Firestore (NoSQL, en tiempo real).
- **Despliegue (Hosting):** Vercel (Configurado para Single Page Application).
- **Estructura Principal (Planificada):**
  - **Dashboard:** Panel central para visualizar el progreso general y un contador regresivo visible para el plazo de 2 meses.
  - **Tarjetas de Estudio (SubjectCards):** Módulos individuales para hacer seguimiento al progreso en Inglés, Cálculo Científico, Seguridad en Redes y Computación en la Nube.
